# 🌸 Perfume Viewer — 3D Experience

An interactive 3D perfume product viewer built with Three.js, designed to showcase a product in a premium, immersive way directly in the browser.

---

## 🔗 Live Demo

👉 https://threejs-perfume-viewer.onrender.com/

---

## ✨ Features

* 💎 Interactive 3D perfume model
* 🔄 Rotate, zoom, and explore from all angles
* 💡 Realistic lighting and materials
* 🎯 Smooth camera controls and user interaction

---

## 🛠️ Tech Stack

* Three.js
* GLTFLoader
* OrbitControls
* Vite
* HTML / CSS / JavaScript

---

## 📌 Notes

* Works in modern browsers with WebGL support
* Designed as a product showcase (e-commerce style experience)
* Three.js enables real-time 3D rendering directly in the browser ([AlterSquare][1])
